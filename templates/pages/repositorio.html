Código Fuente y Despliegue

Accede al núcleo del proyecto, clona el ecosistema y ejecútalo de forma local en simples pasos.

Visitar Repositorio en GitHub

1. Clonar el Proyecto

Abre la terminal o consola de comandos (CMD / PowerShell) en la carpeta donde deseas guardar el proyecto y ejecuta el comando de clonación:

git clone https://github.com/TU_USUARIO/OBJET-X.git

Luego, entra al directorio raíz creado usando el comando: cd OBJET-X

2. Instalar Entorno y Dependencias

Para evitar conflictos de librerías en tu sistema global, se recomienda instalar los paquetes listados en el gestor de requerimientos:

pip install -r requirements.txt
💡 Nota: Este proceso instalará Flask, OpenCV, Pandas y Ultralytics de forma automática.

3. Lanzar la Aplicación

Una vez completada la instalación de los requerimientos, arranca el servidor web asíncrono optimizado para Windows:

python app.py

El sistema desactivará el auto-recargador de PyTorch para evitar bloqueos y dejará el canal de transmisión de video listo.

4. Pruebas y Ajustes

Abre tu navegador web e ingresa a http://localhost:5000. El sistema detectará automáticamente tu webcam interna o USB por defecto.

¿Cómo cambiar los precios en tiempo real?

No necesitas detener el servidor web. Abre tu archivo productos.csv con Microsoft Excel o un editor de texto, cambia el valor de la columna precio o nombre, guarda el archivo y el visor en vivo reflejará los datos modificados al siguiente cuadro detectado.
